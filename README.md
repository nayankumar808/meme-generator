# Meme Generator Web App

## Overview

This project is a Meme Generator web application built using HTML, CSS, and JavaScript. The app fetches random memes from a public API and displays them along with the meme title and author. Users can generate new memes by clicking the "Generate Meme" button.

## Features

- **Random Meme Generation**: Fetches random memes from the [Meme API](https://meme-api.com) and displays them on the screen.
- **Dynamic Content**: Meme title and author are displayed dynamically along with the meme image.
- **Responsive Design**: The application is designed to be responsive and works well on different screen sizes.
- **Interactive User Interface**: Users can generate new memes on demand with a single click.

## Technologies Used

- **HTML**: Structuring the content of the web application.
- **CSS**: Styling the application and ensuring a responsive design.
- **JavaScript**: Handling API requests and dynamically updating the DOM.
